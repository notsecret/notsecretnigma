Ключевая фраза:
Длина строки совпадает с длиной нужной фразы (пробел тоже считается).
количество символов:
Например, для ключевой фразы "пробел тут", пробел будет 7-м символом по порядку, поэтому, для фразы "просто фраза" пробел будет на нужном месте (тоже 7-й символ). А для фразы "вот так" не подходит (пробел 4-м символом, а в ключевой фразе должен быть седьмым).
пробел на нужном месте:
Например, для слова "сок яблочный", будет верно "какие яблочные" тоже (первые 7 букв второго слова одинаковые)
первые 7 букв (или больше) второго слова верно:
Введённая первая буква первого слова совпадает с первой буквой ключевой фразы (которую надо угадать).
первая буква верно:
Во введённом тексте во втором слове первая буква совпадает с первой буквой второго слова ключевой фразы
первая буква второго слова верно:
Последний символ во введённой строке совпадает с последним символом угадываемой фразы.
последняя буква верно:
Не введены "ненужные" символы. Которых нет в ключевой фразе
нет неиспользуемых букв:
Например, для слова "дача" будет верно "чад" и "аааададдчч"(нет лишних букв), но неверно "сдача"(лишняя буква "с") и "чача"(не использована буква "д"). Пробел тоже считается за "букву".
все нужные буквы введены:
Вся фраза угадана верно!
вся фраза правильна:
Великолепно! А это получится? (Удобно на листе бумаги в клеточку)
